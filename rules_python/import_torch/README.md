## Investigate where torch is looking for this lib

1. Go into runfiles dir: `cd bazel-bin/test.runfiles/_main/_test.venv/lib/python3.12/site-packages`
2. See where torch looks for the `libcusparseLt.so.0` with readelf
    ```
    readelf -d torch/_C.cpython-312-x86_64-linux-gnu.so
    # -> torch looks for libtorch_python.so in torch/lib/

    readelf -d torch/lib/libtorch_python.so
    # libtorch_python.so looks for `libcusparseLt.so.0` and many others.
    # libcusparseLt.so.0 is meant to come from $ORIGIN/../../nvidia/cusparselt/lib

    0x0000000000000001 (NEEDED)             Shared library: [libcusparseLt.so.0]
    ```
